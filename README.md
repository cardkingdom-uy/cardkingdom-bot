# cardkingdom-bot
Main bots repository

## discord
cardkingdom-uy discord bot written in python 3

### Installation and running
#### Requirements
* python3
* `pip install -r discord/requirements.txt`
#### Running
* Change current directory to _discord_
* Modify _local_settings.tmp_ to suit your needs and rename it to _local_settings.py_
* Run _bot.py_: `./bot.py` or `python bot.py`
* Have fun!
